# 李宏毅 ML 2022 Lecture 1 笔记

## 机器学习是什么
- 定义：让机器自动找到一个函数 f，输入 x 输出 y
- 和传统编程的区别：传统编程是人写规则，ML 是机器从数据中学规则

## ML 训练的三个步骤
- Step 1：定义模型（带参数的函数）
- Step 2：定义 Loss 函数（衡量预测值和真实值的差距）
- Step 3：优化（找到让 Loss 最小的参数）

## 损失函数 Loss
- 作用：衡量模型好坏，Loss 越低模型越好
- 训练集：用来训练模型，调整参数
- 验证集：用来评估模型，发现过拟合
- 测试集：最终考试，只用一次

## 深度学习 vs 机器学习
- 传统ML：需要人工设计特征
- 深度学习：神经网络自动学习特征
- 鱼与熊掌：深度学习通过好的网络结构，同时做到训练集和测试集 Loss 都低

## 过拟合 Overfitting
- 训练集 Loss 低，但验证集 Loss 高
- 模型只是「记住」了训练数据，没有学到真正的规律
